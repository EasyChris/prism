## 版本 0.1.0（当前）

### 已完成功能

#### 核心功能
- [x] 本地 HTTP 代理服务器
- [x] 配置档案管理（CRUD）
- [x] API Key 加密存储
- [x] 模型 ID 映射
- [x] 请求转发
- [x] SSE 流式响应透传
- [x] 菜单栏集成

#### 日志与审计
- [x] 请求日志记录
- [x] Token 统计（Input/Output/Total）
- [x] 请求耗时监控
- [x] HTTP 状态码记录
- [x] 路由去向追踪
- [x] 日志搜索和过滤

#### 用户界面
- [x] 仪表盘（当前配置详情）
- [x] 配置管理界面
- [x] 日志查看面板
- [x] 统计数据展示
- [x] macOS 菜单栏集成

### 当前限制

- 仅支持 macOS
- 日志保留策略固定（30 天）
- 不支持配置导入/导出
- 不支持自定义主题
- 不支持成本估算

## 版本发布策略

### 发布周期
- **主版本**（Major）：重大架构变更或不兼容更新
- **次版本**（Minor）：新增功能，向后兼容
- **修订版本**（Patch）：Bug 修复和小改进

### 发布流程
1. 功能开发和测试
2. 发布候选版本（RC）
3. 社区测试和反馈
4. 正式发布
5. 发布说明和文档更新

### 更新通知
- GitHub Release
- 应用内更新检查（未来）
- 邮件通知（可选）
